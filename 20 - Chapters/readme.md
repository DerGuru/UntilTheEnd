# Kapitel-Dateinamen (kanonisch)

Die Kapiteldateinamen sind **Identifier** und bleiben numerisch.

Format:

`[7PS-Overall-Position]-[7PS-Position im Abschnitt]-[Kapitelnummer].md`

Kurzform: `X-Y-ZZ.md`

## Bedeutung der Segmente

- `X` = Position in der **7PS overall**
	- `1=Hook`
	- `2=Plot Turn 1`
	- `3=Pinch 1`
	- `4=Midpoint`
	- `5=Pinch 2`
	- `6=Plot Turn 2`
	- `7=Resolution`
- `Y` = 7PS-Position im jeweiligen **Abschnitt**
- `ZZ` = laufende Kapitelnummer innerhalb dieses Slots

Beispiel: `2-1-01.md`
